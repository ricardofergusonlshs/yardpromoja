# YardPromoJa Phase 21 — Homepage Final Hero Tune

This is a CSS-only phase.

```text

docs/PHASE_21_TEST_CHECKLIST.md

## What this phase does

Based on the progress screenshot, this phase tightens the homepage hero so it looks closer to the target reference:

- Keeps the dark premium homepage.
- Tightens the first screen spacing.
- Improves the right-side featured poster/card stack.
- Makes the Featured event label cleaner.
- Makes the View Promo label stronger.
- Helps the search/action area sit better above the fold.
- Improves tablet and phone hero layout.

## Protected

This phase does not touch:

- Supabase keys
- auth/login/signup logic
- admin/dashboard logic
- create promo logic
- browse filters
- link-up logic
- campaign forms/votes/shares
- ad detail JavaScript
- ad detail Open Graph thumbnail logic
- database schema
- homepage JavaScript
- hero carousel logic

## Apply

Copy the included `app` folder into your project root and replace:

```
