# YardPromoJa Phase 22 — Homepage Sections Polish

This is a CSS-only phase.

## Files included

```text
app/yp-dark.css
docs/PHASE_22_TEST_CHECKLIST.md
README_APPLY_PHASE_22_HOMEPAGE_SECTIONS.md
```

## What this phase does

This phase polishes the homepage sections below the hero:

- Featured / premium promo sections
- Campaigns & Sale Offers block
- “What can you promote?” pills
- Popular categories
- Trust band / value tiles
- Business CTA support if present
- Tablet and mobile spacing

## Protected

This phase does not touch:

- Supabase keys
- auth/login/signup logic
- admin/dashboard logic
- create promo logic
- browse filters
- link-up logic
- campaign forms/votes/shares
- ad detail JavaScript
- ad detail Open Graph thumbnail logic
- database schema
- homepage JavaScript
- hero carousel logic
- AdsGrid logic

## Apply

Copy the included `app` folder into your project root and replace:

```text
app/yp-dark.css
```

## Test

Open:

```text
http://localhost:3000
http://localhost:3000/browse
http://localhost:3000/link-up
http://localhost:3000/campaigns
```

Then run:

```cmd
npm run build
```
